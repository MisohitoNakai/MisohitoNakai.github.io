第4条（動作環境）

  1. 本ウェブアプリはモダンブラウザ上での動作を前提としています。
  2. 映像の書き出し機能（MP4エクスポート）は、WebCodecs API に対応したブラウザが必要です。推奨環境は Google Chrome および Microsoft Edge の最新版です。Safari・Firefox では一部機能が利用できない場合があります。
